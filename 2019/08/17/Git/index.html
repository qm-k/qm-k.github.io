git diff

执行 Git diff 以获得更为详细的 git status信息
Git diff 能显示已经写入缓存的数据与未写入缓存的数据之间的差异，主要的应用：

  • 尚未缓存的改动（应该是尚未add的内容，但实际测试出现了所有的改动）：add diff
  • 查看已改动的缓存：git diff --cached
  • 查看已缓存的与未的缓存所有改动：git diff HEAD
  • 显示摘要而非整个diff：git diff stat
